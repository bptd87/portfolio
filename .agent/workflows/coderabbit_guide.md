# CodeRabbit Workflow

1. **Configuration**: The configuration file is located at `coderabbit.yaml` in
   the root directory.
2. **Usage**: CodeRabbit automatically reviews Pull Requests.
   - It provides a high-level summary.
   - It performs line-by-line code review.
   - You can chat with the bot by replying to its comments.

## Commands

- `@coderabbitai pause`: Pause reviews on a PR.
- `@coderabbitai resume`: Resume reviews.
- `@coderabbitai review`: Trigger a manual review.
- `@coderabbitai resolve`: Resolve all open comments.
- `@coderabbitai help`: Show help message.
