# How to Clear Browser Cache

## Chrome/Edge (Windows/Linux):
1. Press `Ctrl + Shift + Delete`
2. Select "Cached images and files"
3. Time range: "All time"
4. Click "Clear data"

OR

1. Press `F12` to open DevTools
2. Right-click the refresh button
3. Select "Empty Cache and Hard Reload"

## Chrome/Edge (Mac):
1. Press `Cmd + Shift + Delete`
2. Select "Cached images and files"
3. Time range: "All time"
4. Click "Clear data"

OR

1. Press `Cmd + Option + I` to open DevTools
2. Right-click the refresh button
3. Select "Empty Cache and Hard Reload"

## Firefox:
1. Press `Ctrl + Shift + Delete` (Windows/Linux) or `Cmd + Shift + Delete` (Mac)
2. Select "Cache"
3. Time range: "Everything"
4. Click "Clear Now"

## Safari (Mac):
1. Press `Cmd + Option + E` to clear cache
2. Or go to Safari > Settings > Advanced > Show Develop menu
3. Then Develop > Empty Caches

## Hard Refresh (Works in all browsers):
- **Windows/Linux**: `Ctrl + Shift + R` or `Ctrl + F5`
- **Mac**: `Cmd + Shift + R`

## Nuclear Option - Disable Cache in DevTools:
1. Open DevTools (`F12` or `Cmd + Option + I`)
2. Go to Network tab
3. Check "Disable cache" checkbox
4. Keep DevTools open while testing
