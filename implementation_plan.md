# Article Focal Point and Gallery Fixes

## Goal Description

Fix two issues reported by the user:

1. **Scenic Insight (Blog) Cover Images**: Support "image focus points" which
   are currently ignored.
2. **Image Gallery**: Allow adding image galleries to blog posts (currently
   failing due to editor/parser mismatch).

## Verification Plan

### Manual Verification

- **Focal Point**:
  1. Go to Admin > Articles > Edit an article.
  2. Set a focal point on the cover image.
  3. Save.
  4. View the article (Scenic Insights) and verify the image positions correctly
     (e.g. check `object-position` style or behavior).
- **Gallery**:
  1. Go to Admin > Articles.
  2. Add a Gallery block using the editor.
  3. Save.
  4. Refresh page (to verify persistence).
  5. View the article and verify gallery renders.
